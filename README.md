colorScroll
=======

https://github.com/hendriklammers/jquery-colorscroll

A jQuery plugin that transitions the background color of an element, based on the scroll position.

Usage
----
```js
$('body').colorScroll({
    colors: [{
        color: '#a2e9ff',
        position: '10%'
    }, {
        color: '#ff92f2',
        position: '50%'
    }, {
        color: '#fff094',
        position: '90%'
    }]
});
```
